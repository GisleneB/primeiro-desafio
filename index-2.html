Estrutura html

H1
O h1 é uma tag, que no html significa o título do texto. Por exemplo, se você colocar o título sem o h1 não vai aparecer no preview.
b
O b é o subtítulo do h1, por exemplo uma continuação do texto.
P
O p significa o parágrafo do texto que você esta escrevendo, se você quer trocar de linha no html tem que colocar o p se não não troca.
Lista
A li é uma tag, que nada menos para fazer uma lista. Tem dois tipos de listas, a lista ordenada (ol) e a lista não ordenada (ul).
